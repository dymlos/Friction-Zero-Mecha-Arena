# Historial

Este directorio preserva la documentacion anterior del proyecto.

## Que se archivo

- Diseno detallado anterior.
- Estados y proximos pasos de la capa activa previa.
- Decisiones tecnicas y ADRs de la estructura intermedia anterior.
- Roadmaps, compatibilidades y snapshots anteriores.
- Planes generados durante sesiones asistidas.

## Como usarlo

- Tomar `docs/` raiz como referencia activa.
- Usar `docs/historial/` cuando haga falta contexto, trazabilidad o wording anterior.
- No asumir que un documento archivado sigue representando la prioridad actual del proyecto.

## Nota de idioma

- El historial puede conservar documentos en su idioma original o con nomenclatura previa.
- La regla de espanol aplica a la capa activa, no necesariamente al archivo historico.
