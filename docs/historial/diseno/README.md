## Regla de mantenimiento

- Estos archivos siguen siendo la fuente de verdad para fantasia, pilares, modos, arquetipos, mapa, UI y restricciones de prototipo.
- Si una decision activa o un roadmap entra en conflicto con este directorio, primero preservar los pilares de `AGENTS.md` y luego actualizar el documento de diseno mas concreto.
- Mantener los archivos tematicos; solo dividir uno si deja de ser comodo de leer por separado.
