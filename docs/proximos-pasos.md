# Proximos pasos

La siguiente iteracion recomendada sigue enfocada en ordenar la base de produccion antes de abrir mas capas.

## Prioridad inmediata

1. Cerrar objetivos reales de escalabilidad, performance y hardware.
2. Definir contratos de input local, ownership de slots, hot-plug y desconexion.
3. Cerrar el contrato de pausa, reanudar y salida segura.
4. Fijar el primer objetivo real de escala de match para `Teams` y `FFA`.
5. Delimitar el alcance inicial de `Modo Practica`.

## Regla de sesion

- Si la sesion es documental, priorizar definiciones de producto.
- Si la sesion es tecnica, no perder disciplina de paridad `base/validation`.
- Si la sesion es de gameplay, no abrir polish audiovisual ni balance fino antes de fijar escalabilidad y escala de match.
