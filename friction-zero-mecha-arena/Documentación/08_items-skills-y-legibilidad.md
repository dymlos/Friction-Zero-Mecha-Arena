# 08 — Ítems, skills y legibilidad

## Sistema de recursos
Se prefiere una solución mixta:
- recarga o munición para una skill propia del robot
- ítems universales del mapa
- algunos ítems de un solo uso, en menor cantidad

## Reglas generales de ítems
- solo se puede llevar un ítem a la vez
- deben verse claros y anticiparse
- deben ser pocos, pero importantes
- aparición semialeatoria

## Tipos de valor preferidos
- munición
- movilidad
- reparación
- energía
- utility

## Legibilidad de skills
- pocos proyectiles o efectos a la vez
- alto nivel de lectura visual
- las skills no deben tapar el combate principal
- pueden empujar o definir jugadas si se usan muy bien

## Principio de claridad
La claridad visual es prioritaria frente al exceso de espectáculo.
