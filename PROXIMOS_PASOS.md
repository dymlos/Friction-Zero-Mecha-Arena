# PROXIMOS_PASOS.md - Friction Zero: Mecha Arena

## Siguiente iteracion recomendada

1. **Cerrar el loop de partes desprendidas**
   - permitir pickup simple
   - asignar propietario original
   - devolver la parte con vida parcial
   - permitir negacion arrojandola al vacio

2. **Agregar cuerpo inutilizado explosivo**
   - temporizador corto
   - empuje radial legible
   - danio moderado a partes cercanas

3. **Dar un primer uso real a la energia**
   - redistribucion discreta
   - cooldown o coste para evitar spam
   - feedback minimo no invasivo

4. **Mejorar validacion jugable**
   - escena o configuracion de prueba para dos jugadores activos
   - ajuste fino de valores de aceleracion, damping, empuje y danio

5. **Preparar soporte Hard sin convertirlo en requisito**
   - torso superior separado
   - rotacion independiente opt-in
   - mantener Easy como modo plenamente jugable
